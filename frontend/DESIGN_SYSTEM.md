# Medical LIS - Design System

## Color Palette

### Primary (Medical Blue)
- 50: #f0f7ff
- 100: #e0effe
- 500: #0c8ce9 (Main)
- 600: #006fc7
- 700: #0158a1

### Secondary (Mint/Teal)
- 50: #f0fdfa
- 100: #ccfbf1
- 500: #14b8a6 (Main)
- 600: #0d9488

### Status Colors
- Success: #22c55e
- Warning: #f59e0b
- Error: #ef4444

## Typography
- Font Family: Inter
- Sizes: 12px (xs), 14px (sm), 16px (base), 18px (lg), 24px (xl), 30px (2xl), 36px (3xl)
- Weights: 300 (light), 400 (normal), 500 (medium), 600 (semibold), 700 (bold)

## Spacing
- Base unit: 4px
- Common: 16px (4), 24px (6), 32px (8), 48px (12)

## Shadows
- soft: 0 1px 3px rgba(0,0,0,0.05)
- soft-lg: 0 4px 6px rgba(0,0,0,0.08)
- soft-xl: 0 10px 15px rgba(0,0,0,0.08)

## Border Radius
- Default: 8px
- Card: 12px
- Large: 16px

## Components

### Buttons
- Primary: Gradient blue, white text
- Secondary: Gray background
- Success: Gradient green
- Sizes: sm (py-2 px-4), md (py-3 px-6), lg (py-4 px-8)

### Cards
- Background: White
- Border radius: 12px
- Shadow: soft
- Padding: 24px

### Inputs
- Border: 1.5px solid gray-200
- Focus: primary-400 border + soft glow
- Padding: 12px 16px
- Icons: 20px, gray-400

### Tables
- Header: gray-50 background
- Hover: gray-50 background
- Border: gray-100

## Icons
- Library: Lucide React
- Size: 20px (default), 24px (large)
- Stroke width: 2 (default), 2.5 (bold)
